# Geometric Survivor - Game Details

## Overview
**Geometric Survivor** is a fast-paced, visually stunning arcade survival game. Players control a geometric entity in a neon-lit arena, fending off waves of increasingly difficult enemies. The game features a polished progression system, dynamic upgrades, and a high-performance custom engine designed for a seamless web experience.

## Game Metadata
- **Game Name:** Geometric Survivor
- **Engine:** Custom HTML5 Canvas (Pure JavaScript, No External Frameworks)
- **Genre:** Survival / Action / Bullet Hell
- **Aesthetic:** Neon Minimalist / Cyberpunk
- **Platforms:** Web (Responsive for Desktop & Mobile)

## Key Features
- **Dynamic Progression:** Level up by collecting XP gems and choose from various upgrades.
- **Upgrades & Power-ups:** Enhance your speed, health, weapon fire rate, or unlock special abilities like shields and magnets.
- **Scaling Difficulty:** Enemies evolve and spawn more frequently as you survive longer.
- **Local Persistence:** High scores and collected currency are saved locally using Web Storage.
- **Audio Experience:** Integrated sound effects for immersive gameplay.

## Controls
### Desktop
- **WASD / Arrow Keys:** Movement
- **Space:** Action / Select / Fire
- **Escape / P:** Pause Game
- **Enter:** Confirm Menu / Restart

### Mobile
- **Virtual Joystick:** Fluid 360-degree movement.
- **Touch Interaction:** Intuitive buttons for menus and selections.
